How to deploy the injectBuild application to EDA

Source location: /afs/btv/data/aes/apps/prod/iipmds (or test)
Target location: /afs/eda/u/einslib/bin/iipmds
  1. In RSA, find a control file called InjectBuild_package_for_EDA.txt in the iipmds/java/src/.../TkInject folder.
  2. Extract that file and InjectBuild_package.pl to any directory (I use $HOME/FOR_EDA)
  3. Cd to the directory you extracted files to (cd $HOME/FOR_EDA)
  4. Run InjectBuild_package.pl.  It will produce a tar file called injectBuild_<YYYYMMDD>.tar
  5. Copy the tar file to /afs/eda/u/einslib/bin
  6. Extract the tar file contents (tar -xvf injectBuild_<YYYYMMDD>.tar)
  7. Cd to /afs/eda/u/einslib/bin/iipmds/bin
  8. Copy injectBuild.last to injectBuild
  9. Cd to /afs/eda/u/einslib/bin
  10. Test the application by running ./injectBuild
